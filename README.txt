== Description ==

Exclude specific post types from Weglot translation.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/exclude-post-type-weglot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Exclude Post Types for Weglot Settings screen to configure the plugin
