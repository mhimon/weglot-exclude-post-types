== Description ==

Exclude specific post types from Weglot translation.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/weglot-exclude-post-type` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Weglot Exclude Post Types Settings screen to configure the plugin
